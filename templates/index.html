Real-Time Prediction

Powered by MediaPipe landmarks mapping, performing geometric tracking at 30 frames per second directly in the client browser with zero lag.

Dual-Speech Engines

Includes client-side Web Speech API and backend Python speech engines. Instantly convert translated text to verbal speech at the push of a button.

Flexible Classification

Switch between a ready-to-use Heuristic rule-based engine and a Scikit-learn Random Forest model trained on hand coordinates.

Keyboard Shortcuts

Control translation without mouse actions. Use Enter to speak, Space to add a space, Backspace to delete, and Esc to clear text.
