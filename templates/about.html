System Architecture

Learn how GestureTalk captures, processes, and translates sign language coordinates in real time.

Real-Time Pipeline

GestureTalk implements an asynchronous local coordinate streaming design that extracts landmarks frame-by-frame on the client-side GPU, sending structured feature vectors back to the Python classification engine.

1

Camera Stream

Camera video captures frames in the client canvas at up to 30 frames per second.

2

Landmark Map

MediaPipe tracks hand joints to extract 21 points in coordinates space.

3

Classification

The engine runs heuristics or machine learning class prediction on the hand vector.

4

Audio Engine

The synthesized text is converted to spoken audio via browser or system TTS.

Technology Stack

Python Backend

  • Flask (Micro web server framework)
  • OpenCV (Local file frames decoding)
  • MediaPipe (Skeletal landmark engine)
  • Scikit-Learn (Random Forest model)
  • Pyttsx3 (Native voice engine fallback)

Frontend Client

  • HTML5 / CSS3 (Inter fonts typography)
  • Vanilla JS (State machine controller)
  • Web Speech Synthesis API (Audio output)
  • Canvas 2D Context API (Draw mapping)
  • MediaPipe Hands JS CDN (GPU accelerated)

MediaPipe Landmarks Reference

The system normalizes the 21 coordinate landmarks of the hand to ensure scale and distance invariance before feeding the 63-value feature list into the ML classifier.

  • 0: Wrist
  • 1: Thumb CMC
  • 2: Thumb MCP
  • 3: Thumb IP
  • 4: Thumb Tip
  • 5: Index MCP
  • 6: Index PIP
  • 7: Index DIP
  • 8: Index Tip
  • 9: Middle MCP
  • 10: Middle PIP
  • 11: Middle DIP
  • 12: Middle Tip
  • 13: Ring MCP
  • 14: Ring PIP
  • 15: Ring DIP
  • 16: Ring Tip
  • 17: Pinky MCP
  • 18: Pinky PIP
  • 19: Pinky DIP
  • 20: Pinky Tip

21 Landmark Points

Normalizes translations based on wrist position (Landmark 0).

Custom Model Training Guide

To record custom gestures and train a custom machine learning model on your own dataset, run the following CLI commands inside your terminal:

# 1. Capture frames from webcam for classes 0 to 7
python collect_data.py
# 2. Train the Random Forest Classifier on captured data
python train_model.py
# 3. Restart server (app loads the new model.p automatically)
python app.py
